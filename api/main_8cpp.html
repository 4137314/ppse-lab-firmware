Detailed Description

Orchestratore di sistema e gestione Multicore.

  • Questo file implementa la logica principale divisa tra Core 0 (UI/Input) e Core 1 (GPS/Storage). La comunicazione avviene tramite code FIFO sicure per il multicore.

◆ loop()

void loop ( )

Loop principale del Core 0.

Gestisce l'interazione in tempo reale:

  1. Aggiorna lo stato dei pulsanti (buttonsUpdate).
  2. Svuota la coda Qdata per ricevere gli ultimi dati GPS dal Core 1.
  3. Salva periodicamente l'ultima posizione valida.
