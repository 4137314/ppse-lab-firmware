Detailed Description

Interfaccia per la gestione del filesystem FatFS e l'integrazione USB Mass Storage.

  • Questo header definisce le funzioni e i flag globali necessari per gestire la memoria flash interna come disco FAT, permettendo la condivisione sicura dei dati tra il firmware dell'RP2040 e un PC esterno.
