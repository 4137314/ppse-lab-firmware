Detailed Description

Implementazione dell'interfaccia utente (UI) su display OLED SSD1306.

  • Gestisce il rendering delle schermate Home, Menu, Meteo e GPS. Include la logica per il cambio di stile del menu (Simple vs Highlight) e il timeout del display.
