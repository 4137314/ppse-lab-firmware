Detailed Description

Definizioni hardware e prototipi per l'interfaccia utente OLED.

  • Questo header definisce la configurazione del display SSD1306 (pin, dimensioni, indirizzi) e fornisce l'interfaccia per il rendering delle diverse schermate applicative.
