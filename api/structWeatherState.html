Detailed Description

Contenitore globale per lo stato meteorologico.

  • Memorizza i dati correnti (temperatura, vento, umidità) e una matrice 7x24 per le previsioni orarie della settimana. Utilizza una maschera di bit per tracciare la ricezione completa dei dati dal server.
