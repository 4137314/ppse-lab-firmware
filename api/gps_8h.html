Detailed Description

Definizioni, macro e strutture per la gestione della telemetria GPS.

  • Questo header definisce l'interfaccia per il modulo GPS, incluse le mappature dei pin UART per RP2040, le impostazioni di logging su filesystem e la struttura dati che aggrega i risultati del parsing della libreria minmea.
